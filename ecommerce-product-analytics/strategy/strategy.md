# E-Commerce Product Analytics Dashboard

## Project Overview

This project analyzes Flipkart laptop data to generate business insights related to:

- product pricing
- discounts
- customer ratings
- brand performance
- laptop specifications
- market segmentation

The goal of this project is to build an end-to-end e-commerce analytics system using Python and Power BI.

---

# Business Problem

E-commerce companies need to understand:

- which products perform better
- how pricing affects customer interest
- which brands dominate the market
- how discounts influence buying behavior
- which laptop specifications impact pricing

This project helps solve these business problems using data analytics and visualization.

---

The main goals of this project are:
